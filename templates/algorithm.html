Algoritma A* & Heuristik H1

Halaman ini khusus membahas konsep A* Search yang dipakai oleh solver, termasuk fungsi evaluasi, heuristik H1, pseudocode, dan ringkasan performa.

01 - Fungsi Evaluasi

f(n) = g(n) + h(n)

f(n) = g(n) + h(n)

g(n) adalah biaya jalur aktual dari state awal ke n, sedangkan h(n) adalah estimasi biaya ke goal. Node dengan nilai f paling kecil diprioritaskan untuk diekspansi.

02 - Heuristik H1

Misplaced Tiles

h1(s) = jumlah ubin salah posisi

H1 menghitung berapa ubin yang belum berada pada posisi goal, tidak termasuk blank (0). Heuristik ini admissible dan konsisten.

03 - Pseudocode

A* Ringkas

A_Star(start, goal):
openSet <- priority queue
while openSet not empty:
  current <- pop f terkecil
  if current == goal: return path
  for each neighbor:
    update g, f, cameFrom
return None
